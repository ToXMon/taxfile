TASK: Schedule B — Interest & Dividends
FILES: taxfile/src/modules/tax-engine/federal/schedule-b.ts
DEPENDS ON: 0022
COMPLEXITY: small

DESCRIPTION:
Schedule B when interest>$1,500 or dividends>$1,500. Lists each payer and amount from 1099-INT/1099-DIV. Sums to totals.

APPROACH:
Pure function from income aggregation. Check thresholds. Structure payer-level detail.

OUTCOME:
Schedule B with payer detail.

ACCEPTANCE CRITERIA:
- Detects when required (>$1,500)
- Lists payers from 1099-INT and 1099-DIV
- Totals match aggregation
- Under 200 lines

DECISIONS LOG:
- Schedule B threshold ($1,500) hardcoded as constant — this is IRS-specified and doesn't change by tax year
- Payer names use document ID prefix since OCR doesn't reliably extract payer names
- When not required, returns empty formLines — caller can check required flag
- Part II dividend line numbers offset by +4 from interest total line — matches actual Schedule B layout
- Qualified dividends stored as 'QD' key (not a real Schedule B line) — informational for tax computation worksheet
- 118 lines, well under 200 limit
- ACCEPTED: all criteria met
